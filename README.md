AI Chatbot with Memory

Project Overview
This project was developed as **Project 1** for the Decode Labs Generative AI Internship.

It is a chatbot built using **Python**, **Streamlit**, and the **Google Gemini API**. The chatbot remembers previous messages during a live session using Streamlit session state.

 Features

- AI-powered chatbot
- Conversation memory
- Clear Chat button
- Simple and interactive user interface


 Technologies Used
 
- Python
- Streamlit
- Google Gemini API

1. Clone this repository.
2. Install the required packages:

pip install -r requirements.txt

3. Add your Gemini API key in `app.py` (or use Streamlit secrets).

4. Run the application:

streamlit run app.py
